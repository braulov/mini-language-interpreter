# Interpreter for an Artificial Programming Language

This project implements a small interpreter in Kotlin for the language described in the test assignment.

The interpreter reads a source program from standard input, executes it, and prints the values of all global variables to standard output.

## Status

Project scaffold and design decisions are being set up first.
Implementation is added incrementally with tests.

## Build

```bash
./gradlew build
```

## Run

```bash
./gradlew run
```
